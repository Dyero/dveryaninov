## 📋 Новый порядок импорта

### 1. Импорт CSV
```bash
1. Откройте Highload блок или Инфоблок
2. Выберите "Импорт" → "Из CSV"
3. Загрузите CSV-файл
4. Настройте соответствие полей (БЕЗ IMAGE_PATH!)
5. Запустите импорт
```

### 2. Загрузка изображений
```bash
1. Откройте список записей
2. Выберите запись для редактирования
3. Найдите поле UF_IMAGE (или IMAGE)
4. Нажмите "Загрузить файл"
5. Выберите изображение из локальной папки images/
6. Сохраните запись
```


## ❓ Быстрые ответы на вопросы

**Q: Нужно ли загружать изображения в /upload/ до импорта?**
A: ❌ НЕТ! Загружайте ПОСЛЕ импорта через форму редактирования.

**Q: Можно ли импортировать всё сразу?**
A: ✅ ДА, но лучше по частям для проверки работоспособности.

**Q: Где взять изображения?**
A: 📂 В локальной папке `images/` вашего проекта.

**Q: Какой формат изображений?**
A: 🖼️ JPG или PNG, рекомендуется 800x800px для покрытий/стёкол, 1200x1600px для дверей.

**Q: Сколько времени займёт загрузка изображений?**
A: ⏱️ При ручной загрузке: 3-4 часа. При использовании скрипта: ~30 минут.

**Q: Есть ли автоматизация?**
A: 🤖 Можно написать PHP-скрипт для массовой загрузки, но требуются навыки программирования.
