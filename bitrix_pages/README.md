# Bitrix Pages - Типовые страницы для загрузки

## 📦 Содержимое папки

Эта папка содержит **готовые к загрузке** страницы для Bitrix CMS:

### Файлы страниц:
1. **about_index.php** — Страница "О компании"
2. **contacts_index.php** — Страница "Контакты" (с картой)
3. **delivery_index.php** — Страница "Доставка и оплата"
4. **guarantee_index.php** — Страница "Гарантия"
5. **privacy_index.php** — Страница "Политика конфиденциальности"

### Документация:
- **БЫСТРЫЙ_СТАРТ.md** — Краткая инструкция (5 минут)
- **ИНСТРУКЦИЯ_ПО_ЗАГРУЗКЕ.md** — Подробная инструкция со всеми деталями

---

## 🚀 Как использовать

### Вариант 1: Быстрый (рекомендуется)
Откройте файл **БЫСТРЫЙ_СТАРТ.md** и следуйте инструкциям.

### Вариант 2: Подробный
Откройте файл **ИНСТРУКЦИЯ_ПО_ЗАГРУЗКЕ.md** для детальной инструкции.

---

## 📝 Что нужно сделать

1. Создать 5 папок в Bitrix: `/about`, `/contacts`, `/delivery`, `/guarantee`, `/privacy`
2. В каждую папку загрузить соответствующий файл как `index.php`
3. Загрузить изображения для страницы "О компании"
4. Очистить кеш Bitrix
5. Проверить все страницы в браузере

**Время выполнения:** ~15-20 минут

---

## ✅ Что уже готово

Все файлы **полностью адаптированы** под Bitrix:
- ✅ Используют `header.php` и `footer.php` из шаблона
- ✅ Пути к изображениям через `SITE_TEMPLATE_PATH`
- ✅ Мета-теги (title, description) настроены
- ✅ Хлебные крошки работают
- ✅ Ссылки изменены с `.html` на `/`
- ✅ Удалены лишние блоки (mobile menu — он в footer.php)

**Никаких изменений в коде делать не нужно!** Просто загрузите как есть.

## 🆘 Помощь

Если что-то не получается:
1. Откройте **ИНСТРУКЦИЯ_ПО_ЗАГРУЗКЕ.md** → раздел "Возможные проблемы"
