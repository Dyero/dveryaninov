# Краткая инструкция: Загрузка типовых страниц в Bitrix

## Быстрый старт

### 1. Создать папки
В **Контент → Файловая структура** создайте:
- `/about`
- `/contacts`
- `/delivery`
- `/guarantee`
- `/privacy`

### 2. Загрузить файлы

| Папка | Файл для загрузки | Итоговое имя |
|-------|-------------------|--------------|
| `/about/` | `about_index.php` | `index.php` |
| `/contacts/` | `contacts_index.php` | `index.php` |
| `/delivery/` | `delivery_index.php` | `index.php` |
| `/guarantee/` | `guarantee_index.php` | `index.php` |
| `/privacy/` | `privacy_index.php` | `index.php` |

### 3. Способ загрузки

**Вариант A: Через "Создать файл"**
1. Открыть нужную папку
2. Нажать "Создать файл" → Имя: `index.php`
3. Скопировать код из соответствующего файла
4. Вставить в редактор Bitrix
5. Сохранить

**Вариант B: Через "Загрузить файл"**
1. Открыть нужную папку
2. Нажать "Загрузить файл"
3. Выбрать файл (например, `about_index.php`)
4. После загрузки переименовать в `index.php`

### 4. Загрузить изображения для страницы "О компании"

В `/local/templates/dveryaninov/images/` загрузите:
- `hero-bg.svg`
- `news-1.png`, `news-2.png`, `news-3.png`, `news-4.png`
- `card-door-1.svg`, `card-door-2.svg`, `card-door-3.svg`, `card-door-4.svg`

### 5. Очистить кеш
**Настройки → Производительность → Управление кешем → Очистить весь кеш**

### 6. Проверить
- `https://ваш-сайт.ru/about/`
- `https://ваш-сайт.ru/contacts/`
- `https://ваш-сайт.ru/delivery/`
- `https://ваш-сайт.ru/guarantee/`
- `https://ваш-сайт.ru/privacy/`

---

## Важно!

✅ Файлы должны называться **index.php** (не `about_index.php`)
✅ Кодировка файлов: **UTF-8 без BOM**
✅ После загрузки обязательно **очистить кеш**
✅ Изображения загружать только для страницы "О компании"

---

## Если что-то не работает

1. **404 ошибка** → Проверьте имя файла (`index.php`) и очистите кеш
2. **Нет стилей** → Проверьте загрузку CSS в `/local/templates/dveryaninov/css/`
3. **Кракозябры** → Пересохраните файл в UTF-8 без BOM
4. **Нет картинок** → Загрузите их в `/local/templates/dveryaninov/images/`
