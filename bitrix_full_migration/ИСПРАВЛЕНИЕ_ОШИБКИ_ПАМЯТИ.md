# 🔴 КРИТИЧЕСКОЕ ИСПРАВЛЕНИЕ: Fatal error memory exhausted

## ПРОБЛЕМА

Ошибка: `Fatal error: Allowed memory size of 268435456 bytes exhausted`

**Причина:** Использование компонента `bitrix:main.include` для подключения header.php и footer.php создаёт бесконечную рекурсию!

## ✅ РЕШЕНИЕ

Заменить `$APPLICATION->IncludeComponent("bitrix:main.include", ...)` на обычный `include()`.

---

## ИСПРАВЛЕННЫЕ ФАЙЛЫ

### 1. Исправить `/local/templates/dveryaninov/header.php`

**Было (строки 30-41):**
```php
<?php
// Include site header component
$APPLICATION->IncludeComponent(
    "bitrix:main.include",
    "",
    array(
        "AREA_FILE_SHOW" => "file",
        "PATH" => SITE_TEMPLATE_PATH . "/includes/header.php",
        "EDIT_TEMPLATE" => ""
    )
);
?>
```

**ЗАМЕНИТЬ НА:**
```php
<?php
// Include site header
$includeHeaderPath = $_SERVER["DOCUMENT_ROOT"] . SITE_TEMPLATE_PATH . "/includes/header.php";
if (file_exists($includeHeaderPath)) {
    include($includeHeaderPath);
}
?>
```

---

### 2. Исправить `/local/templates/dveryaninov/footer.php`

**Было (строки 5-16):**
```php
<?php
// Include site footer component
$APPLICATION->IncludeComponent(
    "bitrix:main.include",
    "",
    array(
        "AREA_FILE_SHOW" => "file",
        "PATH" => SITE_TEMPLATE_PATH . "/includes/footer.php",
        "EDIT_TEMPLATE" => ""
    )
);
?>
```

**ЗАМЕНИТЬ НА:**
```php
<?php
// Include site footer
$includeFooterPath = $_SERVER["DOCUMENT_ROOT"] . SITE_TEMPLATE_PATH . "/includes/footer.php";
if (file_exists($includeFooterPath)) {
    include($includeFooterPath);
}
?>
```

## ПОШАГОВАЯ ИНСТРУКЦИЯ ИСПРАВЛЕНИЯ

### Шаг 1: Исправить header.php

1. Откройте: **Контент → Файловая структура → /local/templates/dveryaninov/header.php**
2. Найдите строки с `$APPLICATION->IncludeComponent("bitrix:main.include", ...`
3. Замените их на:
```php
<?php
// Include site header
$includeHeaderPath = $_SERVER["DOCUMENT_ROOT"] . SITE_TEMPLATE_PATH . "/includes/header.php";
if (file_exists($includeHeaderPath)) {
    include($includeHeaderPath);
}
?>
```
4. Сохраните файл

### Шаг 2: Исправить footer.php

1. Откройте: **Контент → Файловая структура → /local/templates/dveryaninov/footer.php**
2. Найдите строки с `$APPLICATION->IncludeComponent("bitrix:main.include", ...`
3. Замените их на:
```php
<?php
// Include site footer
$includeFooterPath = $_SERVER["DOCUMENT_ROOT"] . SITE_TEMPLATE_PATH . "/includes/footer.php";
if (file_exists($includeFooterPath)) {
    include($includeFooterPath);

?>
```
4. Сохраните файл

### Шаг 3: Очистить кеш

1. **Настройки → Производительность → Управление кешем**
2. Нажмите **"Очистить весь кеш"**

### Шаг 4: Проверить

1. Обновите страницу сайта
2. Ошибка должна исчезнуть
3. Сайт должен работать нормально

## ПОЧЕМУ ЭТО РАБОТАЕТ?

1. **bitrix:main.include** - это компонент для подключения контентных областей на странице
2. Он НЕ предназначен для подключения частей шаблона (header/footer)
3. Для шаблонов используется обычный `include()` или `require()`
4. Это избегает рекурсии и излишней обработки

---

## АЛЬТЕРНАТИВНЫЙ ВАРИАНТ (ещё проще)

Можно использовать ещё более простой вариант:

### header.php:
```php
<?php include($_SERVER["DOCUMENT_ROOT"] . SITE_TEMPLATE_PATH . "/includes/header.php"); ?>
```

### footer.php:
```php
<?php include($_SERVER["DOCUMENT_ROOT"] . SITE_TEMPLATE_PATH . "/includes/footer.php"); ?>
```

**Но лучше использовать вариант с проверкой `file_exists()` для безопасности!**
