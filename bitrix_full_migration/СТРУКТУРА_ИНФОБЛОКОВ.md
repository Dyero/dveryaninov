# Структура инфоблоков системы

## Инфоблок "Дверные полотна" (ID: 6)

**Назначение:** Каталог межкомнатных дверей

**Основные свойства:**
- Коллекция, номер модели
- Тип полотна (ПГ/ПО) с множественным выбором
- Базовые цены для ПГ и ПО отдельно
- Параметры двери (высота, покрытия, гравировка)
- Привязка к декоративным молдингам (через HL-блок)

**Подробная документация:** См. файл `СТРУКТУРА_СВОЙСТВ_ИНФОБЛОКА_ДВЕРИ.md`

## Инфоблок "Погонаж" (ID: 5)

**Назначение:** Каталог погонажных изделий (продаются погонными метрами)

**Что входит:**
- Наличники
- Доборы
- Карнизы
- Плинтусы
- Прочие погонажные изделия

### Таблица свойств погонажа

| Код свойства        | Название                    | Тип                      | Множественное | Обязательное | Примечание |
|--------------------|-----------------------------|--------------------------|---------------|--------------|------------|
| PRODUCT_TYPE        | Тип изделия                 | Список                   | Нет           | Да           | Наличник, Добор, Карниз, Плинтус |
| COLLECTION          | Коллекция                   | Строка                   | Нет           | Нет          | Название коллекции (для совместимости с дверями) |
| WIDTH               | Ширина (мм)                 | Число                    | Нет           | Нет          | Ширина изделия в миллиметрах |
| HEIGHT              | Высота/Толщина (мм)         | Число                    | Нет           | Нет          | Высота или толщина изделия |
| PRICE_PER_METER     | Цена за погонный метр       | Число                    | Нет           | Да           | Цена за 1 погонный метр |
| MIN_LENGTH          | Минимальная длина (м)       | Число                    | Нет           | Нет          | Минимальная длина заказа |
| MAX_LENGTH          | Максимальная длина (м)      | Число                    | Нет           | Нет          | Максимальная длина изделия |
| COATING_TYPES       | Допустимые покрытия         | Строка                   | Нет           | Нет          | Список допустимых покрытий |
| MATERIAL            | Материал                    | Список                   | Нет           | Нет          | МДФ, Массив, ПВХ и т.д. |
| COLOR_MATCH         | Совместимость по цвету      | Строка                   | Нет           | Нет          | С какими цветами дверей сочетается |
| IMAGE               | Изображение                 | Файл                     | Нет           | Нет          | Основное изображение изделия |

### Список значений для PRODUCT_TYPE

- `NALICHNIK` → "Наличник"
- `DOBOR` → "Добор"
- `KARNIZ` → "Карниз"
- `PLINTUS` → "Плинтус"
- `OTHER` → "Прочее"

### Примеры использования в коде

```php
// Получение погонажных изделий для конкретной коллекции
$rsItems = CIBlockElement::GetList(
    ["SORT" => "ASC", "NAME" => "ASC"],
    [
        "IBLOCK_ID" => 5, // ID инфоблока "Погонаж"
        "ACTIVE" => "Y",
        "PROPERTY_COLLECTION" => "Альберта" // Фильтр по коллекции
    ],
    false,
    false,
    ["ID", "NAME", "PROPERTY_PRODUCT_TYPE", "PROPERTY_PRICE_PER_METER", "PROPERTY_WIDTH"]
);

while ($arItem = $rsItems->GetNext()) {
    echo $arItem["NAME"] . " - " . $arItem["PROPERTY_PRODUCT_TYPE_VALUE"];
    echo " (" . number_format($arItem["PROPERTY_PRICE_PER_METER_VALUE"], 0) . " ₽/м)";
}
```

### Расчёт стоимости погонажа

```php
// Пользователь выбирает длину в метрах
$selectedLength = 2.5; // метры
$pricePerMeter = $arItem["PROPERTY_PRICE_PER_METER_VALUE"];

// Итоговая стоимость
$totalPrice = $selectedLength * $pricePerMeter;

echo "Стоимость: " . number_format($totalPrice, 0) . " ₽";
```

## Highload-блок "Декоративные молдинги" (будет создан позже)

**Назначение:** Справочник декоративных накладных элементов для дверных полотен

**Что входит:**
- Декоративные накладки
- Рамки
- Орнаменты
- Прочие декоративные элементы, устанавливаемые на полотно

### Планируемая структура полей

| Поле                | Тип                      | Примечание |
|--------------------|--------------------------|------------|
| UF_NAME             | Строка                   | Название молдинга |
| UF_CODE             | Строка                   | Символьный код |
| UF_PRICE            | Число                    | Стоимость молдинга |
| UF_IMAGE            | Файл                     | Изображение |
| UF_COMPATIBLE_DOORS | Привязка к инфоблоку     | С какими дверями совместим |
| UF_POSITION         | Строка                   | Расположение на двери (верх, низ, центр) |
| UF_SIZE             | Строка                   | Размеры молдинга |

### Связь с дверями

В инфоблоке "Дверные полотна" (ID: 6) есть свойство **DECORATIVE_MOLDINGS** типа "Привязка к элементам HL-блока", которое будет связывать двери с доступными молдингами.

## Структура директорий для страниц

```
/catalog/                          → Каталог дверей (инфоблок ID: 6)
  └── index.php                    → catalog_index.php с компонентом doors.catalog
  └── [DOOR_CODE]/                 → Детальная страница двери
      └── index.php                → product_detail.php

/pogonazh/                         → Каталог погонажа (инфоблок ID: 5)
  └── index.php                    → Список погонажных изделий
  └── [ITEM_CODE]/                 → Детальная страница погонажного изделия
      └── index.php

/collections/                      → Коллекции
  └── index.php                    → collections_index.php
  └── [COLLECTION_CODE]/           → Страница коллекции
      └── index.php
```

## Компоненты Bitrix

### Для дверей (инфоблок ID: 6)
- `dveryaninov:doors.catalog` — каталог дверей с фильтрами
- `bitrix:catalog.section` — список дверей в разделе
- `bitrix:catalog.element` — детальная страница двери

### Для погонажа (инфоблок ID: 5)
- `bitrix:catalog.section` — список погонажа
- `bitrix:catalog.element` — детальная страница погонажного изделия

### Для молдингов (HL-блок)
- Пользовательский компонент с использованием `Bitrix\Highloadblock\HighloadBlockTable`
