# ПОЛНАЯ МИГРАЦИЯ НА BITRIX - ПЛАН И РЕШЕНИЕ ПРОБЛЕМ

## 🔴 ТЕКУЩИЕ ПРОБЛЕМЫ

### 1. Главная страница не показывается
**Причина:** Нет файла `/index.php` в корне сайта Bitrix

### 2. Каталог не подгружает товары из инфоблоков
**Причина:** Компонент `doors.catalog` не подключён к инфоблокам корректно

### 3. Header и footer выглядят неправильно
**Причина:**
- Отсутствуют JS файлы для слайдеров и интерактивности
- Не подключён попап авторизации
- Нет бургер-меню логики

### 4. Нет интеграции авторизации
**Причина:** auth.js использует localStorage, но не интегрирован с Bitrix USER

### 5. Нет корзины
**Причина:** Не создана страница `/personal/cart/` и не интегрирована с Bitrix Sale

### ЭТАП 5: Интеграция авторизации

**Проблема:** auth.js работает через localStorage, но Bitrix использует свою систему

**Решение:** Модифицировать `auth.js` для работы с Bitrix USER:

**Файл:** `/local/templates/dveryaninov/js/auth-bitrix.js`

```javascript
// Проверка авторизации через Bitrix
function checkBitrixAuth() {
    // Bitrix автоматически устанавливает глобальную переменную USER
    return typeof BX !== 'undefined' && BX.message('USER_ID') > 0;
}

// Открыть попап авторизации
function openAuthPopup() {
    // Используем стандартный компонент Bitrix в попапе
    var popup = new BX.PopupWindow('auth-popup', null, {
        content: BX('bitrix-auth-form'),
        closeIcon: true,
        width: 400,
        height: 300
    });
    popup.show();
}

// Выход
function logout() {
    window.location.href = '/auth/?logout=yes&sessid=' + BX.bitrix_sessid();
}
```

**Добавить в header.php скрытую форму авторизации:**

```php
<?php if (!$USER->IsAuthorized()): ?>
<div id="bitrix-auth-form" style="display:none;">
    <?$APPLICATION->IncludeComponent(
        "bitrix:system.auth.form",
        "popup",
        array(
            "REGISTER_URL" => "/auth/registration/",
            "FORGOT_PASSWORD_URL" => "/auth/forgot-password/",
            "PROFILE_URL" => "/personal/",
            "SHOW_ERRORS" => "Y",
        )
    );?>
</div>
<?php endif; ?>
```

```php
// Фильтр по коллекции
$arFilter = ["IBLOCK_ID" => $arParams["IBLOCK_ID"], "ACTIVE" => "Y"];

if (!empty($_REQUEST["collection"])) {
    $collectionCode = htmlspecialchars($_REQUEST["collection"]);
    // Найти ID коллекции по символьному коду
    $rsCollection = CIBlockElement::GetList(
        [],
        ["IBLOCK_ID" => 1, "CODE" => $collectionCode, "ACTIVE" => "Y"],
        false,
        false,
        ["ID"]
    );
    if ($arCollection = $rsCollection->Fetch()) {
        $arFilter["PROPERTY_COLLECTION"] = $arCollection["ID"];
    }
}

// Получить товары
$rsDoors = CIBlockElement::GetList(
    ["SORT" => "ASC", "NAME" => "ASC"],
    $arFilter,
    false,
    ["nPageSize" => $arParams["ELEMENTS_COUNT"]],
    ["ID", "NAME", "CODE", "PREVIEW_PICTURE", "PROPERTY_PRICE", "PROPERTY_COLLECTION"]
);
```
