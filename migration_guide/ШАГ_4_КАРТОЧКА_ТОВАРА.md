# Шаг 4: Карточка товара (детальная страница)

## Цель этапа

Создать детальную страницу двери с возможностью выбора типа полотна (ПГ/ПО) и отображением цены.

---

## 4.1 Настройка ЧПУ (красивых URL)

### В админ-панели Bitrix:

1. Перейдите: **Настройки → Настройки продукта → Настройки модулей → Главный модуль**
2. Вкладка **"ЧПУ"**
3. Добавьте правило:
   - **Условие:** `#^/catalog/([^/]+)/$#`
   - **Путь:** `/catalog/detail.php?CODE=$1`
   - **Компонент:** (оставьте пустым или выберите catalog.element)
4. Сохраните

---

## 4.2 Создание файла детальной страницы

### 📄 Файл: `/home/bitrix/www/catalog/detail.php`

```php
<?php
require($_SERVER["DOCUMENT_ROOT"]."/bitrix/header.php");

// Получаем символьный код товара из URL
$doorCode = $_REQUEST['CODE'] ?? '';

if (!$doorCode) {
    LocalRedirect("/catalog/");
}

// Получаем информацию о двери
$rsDoor = CIBlockElement::GetList(
    [],
    [
        "IBLOCK_ID" => 6,
        "CODE" => $doorCode,
        "ACTIVE" => "Y"
    ],
    false,
    false,
    [
        "ID",
        "NAME",
        "CODE",
        "DETAIL_TEXT",
        "DETAIL_PICTURE",
        "PREVIEW_PICTURE",
        "PROPERTY_COLLECTION",
        "PROPERTY_MODEL_NUMBER",
        "PROPERTY_BLADE_TYPE",
        "PROPERTY_BASE_PRICE_PG",
        "PROPERTY_BASE_PRICE_PO",
        "PROPERTY_MAX_HEIGHT"
    ]
);

$arDoor = $rsDoor->GetNext();

if (!$arDoor) {
    LocalRedirect("/catalog/");
}

// Получаем свойства
$doorId = $arDoor["ID"];
$rsProps = CIBlockElement::GetProperty($arDoor["IBLOCK_ID"], $doorId);
$arProps = [];
while ($arProp = $rsProps->Fetch()) {
    if ($arProp["MULTIPLE"] == "Y") {
        $arProps[$arProp["CODE"]][] = $arProp["VALUE"];
    } else {
        $arProps[$arProp["CODE"]] = $arProp["VALUE"];
    }
}

// Доступные типы полотна
$bladeTypes = $arProps["BLADE_TYPE"] ?? [];
if (!is_array($bladeTypes)) {
    $bladeTypes = [$bladeTypes];
}

$APPLICATION->SetTitle($arDoor["NAME"]);
?>

<?php
// Подключаем шапку
$includeHeaderPath = $_SERVER["DOCUMENT_ROOT"] . SITE_TEMPLATE_PATH . "/includes/header.php";
if (file_exists($includeHeaderPath)) {
    include($includeHeaderPath);
}
?>

<main class="product-page">
    <nav class="breadcrumbs">
        <a href="/">Главная</a>
        <span>–</span>
        <a href="/catalog/">Каталог</a>
        <span>–</span>
        <span><?= htmlspecialcharsbx($arDoor["NAME"]) ?></span>
    </nav>

    <h1><?= htmlspecialcharsbx($arDoor["NAME"]) ?></h1>

    <div class="product">
        <!-- Изображение -->
        <div class="product__image">
            <?php
            $imageSrc = $arDoor["DETAIL_PICTURE"]
                ? CFile::GetPath($arDoor["DETAIL_PICTURE"])
                : CFile::GetPath($arDoor["PREVIEW_PICTURE"]);
            ?>
            <img src="<?= $imageSrc ?>" alt="<?= htmlspecialcharsbx($arDoor["NAME"]) ?>">
        </div>

        <!-- Информация -->
        <div class="product__info">

            <!-- Выбор типа полотна -->
            <?php if (count($bladeTypes) > 0): ?>
            <div class="product__option">
                <label>Тип полотна:</label>
                <div class="product__variants">
                    <?php foreach ($bladeTypes as $type): ?>
                    <label class="variant-option">
                        <input type="radio" name="blade_type" value="<?= htmlspecialcharsbx($type) ?>"
                               data-price-pg="<?= $arDoor["PROPERTY_BASE_PRICE_PG_VALUE"] ?? 0 ?>"
                               data-price-po="<?= $arDoor["PROPERTY_BASE_PRICE_PO_VALUE"] ?? 0 ?>"
                               <?= $type === $bladeTypes[0] ? 'checked' : '' ?>>
                        <span><?= htmlspecialcharsbx($type) ?></span>
                    </label>
                    <?php endforeach; ?>
                </div>
            </div>
            <?php endif; ?>

            <!-- Цена -->
            <div class="product__price">
                <span id="current-price">
                    <?php
                    // Определяем начальную цену
                    $defaultType = $bladeTypes[0] ?? 'ПГ';
                    $initialPrice = ($defaultType === 'ПГ')
                        ? ($arDoor["PROPERTY_BASE_PRICE_PG_VALUE"] ?? 0)
                        : ($arDoor["PROPERTY_BASE_PRICE_PO_VALUE"] ?? 0);
                    ?>
                    от <?= number_format($initialPrice, 0, ',', ' ') ?> ₽
                </span>
            </div>

            <!-- Кнопка в корзину -->
            <button class="btn btn_primary" onclick="addToCart(<?= $doorId ?>)">
                Добавить в корзину
            </button>

            <!-- Описание -->
            <?php if ($arDoor["DETAIL_TEXT"]): ?>
            <div class="product__description">
                <h2>Описание</h2>
                <?= $arDoor["DETAIL_TEXT"] ?>
            </div>
            <?php endif; ?>
        </div>
    </div>
</main>

<script>
// Обновление цены при смене типа полотна
document.querySelectorAll('input[name="blade_type"]').forEach(function(radio) {
    radio.addEventListener('change', function() {
        var pricePG = parseInt(this.dataset.pricePg);
        var pricePO = parseInt(this.dataset.pricePo);
        var price = this.value === 'ПГ' ? pricePG : pricePO;

        document.getElementById('current-price').textContent =
            'от ' + price.toLocaleString('ru-RU') + ' ₽';
    });
});
</script>

<?php
// Подключаем футер
$includeFooterPath = $_SERVER["DOCUMENT_ROOT"] . SITE_TEMPLATE_PATH . "/includes/footer.php";
if (file_exists($includeFooterPath)) {
    include($includeFooterPath);
}
?>

<?php
require($_SERVER["DOCUMENT_ROOT"]."/bitrix/footer.php");
?>
```

- Хлебные крошки работают
