# Шаг 2: Главная страница

## Цель этапа

Создать главную страницу сайта, которая будет отображать контент из инфоблоков.

---

## 2.1 Создание файла главной страницы

### 📄 Создайте файл: `/home/bitrix/www/index.php`

**Содержимое файла:**

```php
<?php
require($_SERVER["DOCUMENT_ROOT"]."/bitrix/header.php");
$APPLICATION->SetTitle("Дверянинов — Премиальные межкомнатные двери");
$APPLICATION->SetPageProperty("description", "Премиальные межкомнатные двери от фабрики Дверянинов. Собственное производство, широкий выбор коллекций, индивидуальные размеры.");
?>

<?php
// Подключаем шапку сайта
$includeHeaderPath = $_SERVER["DOCUMENT_ROOT"] . SITE_TEMPLATE_PATH . "/includes/header.php";
if (file_exists($includeHeaderPath)) {
    include($includeHeaderPath);
}
?>

<main>
    <!-- Здесь будет контент главной страницы -->
    <h1>Добро пожаловать!</h1>
    <p>Главная страница работает</p>
</main>

<?php
// Подключаем футер сайта
$includeFooterPath = $_SERVER["DOCUMENT_ROOT"] . SITE_TEMPLATE_PATH . "/includes/footer.php";
if (file_exists($includeFooterPath)) {
    include($includeFooterPath);
}
?>

<?php
require($_SERVER["DOCUMENT_ROOT"]."/bitrix/footer.php");
?>
```

## 2.3 Добавление контента с главной HTML-страницы

### Откройте исходный файл `index.html` из репозитория

**Скопируйте блоки контента и адаптируйте их:**

### Пример адаптации Hero-блока:

**Из `index.html`:**
```html
<img src="images/news-1.png" alt="...">
```

**В `index.php`:**
```php
<img src="<?= SITE_TEMPLATE_PATH ?>/images/news-1.png" alt="...">
```

### Пример блока с данными из инфоблока:

**Вместо статичного HTML:**
```html
<article class="card">
    <img src="images/door1.jpg">
    <h3>Дверь 1</h3>
</article>
```

**Используйте динамический вывод:**
```php
<?php
// Получаем 4 двери из инфоблока
$rsDoors = CIBlockElement::GetList(
    ["SORT" => "ASC"],
    [
        "IBLOCK_ID" => 6, // ID инфоблока дверей
        "ACTIVE" => "Y"
    ],
    false,
    ["nTopCount" => 4],
    ["ID", "NAME", "CODE", "PREVIEW_PICTURE", "PROPERTY_BASE_PRICE_PG"]
);

while ($arDoor = $rsDoors->GetNext()) {
    $imageSrc = CFile::GetPath($arDoor["PREVIEW_PICTURE"]);
    $price = $arDoor["PROPERTY_BASE_PRICE_PG_VALUE"] ?? 0;
    ?>
    <article class="card">
        <img src="<?= $imageSrc ?>" alt="<?= htmlspecialcharsbx($arDoor["NAME"]) ?>">
        <h3><a href="/catalog/<?= $arDoor["CODE"] ?>/"><?= htmlspecialcharsbx($arDoor["NAME"]) ?></a></h3>
        <p>от <?= number_format($price, 0, ',', ' ') ?> ₽</p>
    </article>
    <?php
}
?>
```

## 2.4 Структура главной страницы

### Рекомендуемые блоки:

1. **Hero-блок** — большой баннер с призывом к действию
2. **Популярные двери** — 4-6 дверей из инфоблока
3. **О компании** — краткая информация
4. **Преимущества** — иконки с описанием
5. **Контакты** — форма обратной связи

### Используйте готовые блоки из `index.html`

Просто замените пути к изображениям:
- `images/` → `<?= SITE_TEMPLATE_PATH ?>/images/`
- `css/` → `<?= SITE_TEMPLATE_PATH ?>/css/`
- `js/` → `<?= SITE_TEMPLATE_PATH ?>/js/`
