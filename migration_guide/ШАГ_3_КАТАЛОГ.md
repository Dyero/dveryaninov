# Шаг 3: Страница каталога

## Цель этапа

Создать страницу каталога дверей с фильтрами и выводом товаров из инфоблока.

---

## 3.1 Создание структуры каталога

### Создайте папку и файл на сервере:

```
/home/bitrix/www/catalog/
└── index.php
```

---

## 3.2 Создание файла каталога

### 📄 Файл: `/home/bitrix/www/catalog/index.php`

**Простая версия (без компонента):**

```php
<?php
require($_SERVER["DOCUMENT_ROOT"]."/bitrix/header.php");
$APPLICATION->SetTitle("Каталог межкомнатных дверей");
?>

<?php
// Подключаем шапку
$includeHeaderPath = $_SERVER["DOCUMENT_ROOT"] . SITE_TEMPLATE_PATH . "/includes/header.php";
if (file_exists($includeHeaderPath)) {
    include($includeHeaderPath);
}
?>

<main class="catalog-page">
    <nav class="breadcrumbs">
        <a href="/">Главная</a>
        <span>–</span>
        <span>Каталог</span>
    </nav>

    <h1>Каталог дверей</h1>

    <div class="catalog__grid">
        <?php
        // Получаем все двери из инфоблока
        $rsDoors = CIBlockElement::GetList(
            ["SORT" => "ASC", "NAME" => "ASC"],
            [
                "IBLOCK_ID" => 6, // ID инфоблока дверей
                "ACTIVE" => "Y"
            ],
            false,
            ["nPageSize" => 20], // Пагинация: 20 товаров на странице
            [
                "ID",
                "NAME",
                "CODE",
                "PREVIEW_PICTURE",
                "PROPERTY_BASE_PRICE_PG",
                "PROPERTY_BASE_PRICE_PO",
                "PROPERTY_BLADE_TYPE",
                "PROPERTY_COLLECTION"
            ]
        );

        while ($arDoor = $rsDoors->GetNext()) {
            $imageSrc = CFile::GetPath($arDoor["PREVIEW_PICTURE"]);
            $pricePG = $arDoor["PROPERTY_BASE_PRICE_PG_VALUE"] ?? 0;
            $pricePO = $arDoor["PROPERTY_BASE_PRICE_PO_VALUE"] ?? 0;

            // Определяем минимальную цену
            $minPrice = $pricePG > 0 ? $pricePG : $pricePO;
            if ($pricePG > 0 && $pricePO > 0) {
                $minPrice = min($pricePG, $pricePO);
            }
            ?>
            <article class="card">
                <div class="card__image-wrap">
                    <img class="card__image" src="<?= $imageSrc ?>" alt="<?= htmlspecialcharsbx($arDoor["NAME"]) ?>">
                </div>
                <div class="card__info">
                    <h3 class="card__title">
                        <a href="/catalog/<?= $arDoor["CODE"] ?>/"><?= htmlspecialcharsbx($arDoor["NAME"]) ?></a>
                    </h3>
                    <p class="card__price">от <?= number_format($minPrice, 0, ',', ' ') ?> ₽</p>
                </div>
            </article>
            <?php
        }
        ?>
    </div>

    <?php
    // Пагинация
    echo $rsDoors->GetPageNavStringEx();
    ?>
</main>

<?php
// Подключаем футер
$includeFooterPath = $_SERVER["DOCUMENT_ROOT"] . SITE_TEMPLATE_PATH . "/includes/footer.php";
if (file_exists($includeFooterPath)) {
    include($includeFooterPath);
}
?>

<?php
require($_SERVER["DOCUMENT_ROOT"]."/bitrix/footer.php");
?>
```

## 3.3 Добавление фильтров (опционально)

### Если нужны фильтры по коллекциям:

```php
<?php
// Получаем выбранную коллекцию из GET-параметра
$selectedCollection = $_GET['collection'] ?? '';

// Формируем фильтр
$arFilter = [
    "IBLOCK_ID" => 6,
    "ACTIVE" => "Y"
];

if ($selectedCollection) {
    $arFilter["PROPERTY_COLLECTION"] = $selectedCollection;
}

// Получаем двери с фильтром
$rsDoors = CIBlockElement::GetList(
    ["SORT" => "ASC", "NAME" => "ASC"],
    $arFilter,
    false,
    ["nPageSize" => 20],
    ["ID", "NAME", "CODE", "PREVIEW_PICTURE", "PROPERTY_BASE_PRICE_PG"]
);
?>

<!-- HTML фильтра -->
<div class="catalog__filters">
    <a href="/catalog/" class="filter-btn <?= !$selectedCollection ? 'active' : '' ?>">Все</a>
    <a href="/catalog/?collection=Альберта" class="filter-btn <?= $selectedCollection === 'Альберта' ? 'active' : '' ?>">Альберта</a>
    <a href="/catalog/?collection=Флай" class="filter-btn <?= $selectedCollection === 'Флай' ? 'active' : '' ?>">Флай</a>
    <!-- И так далее для других коллекций -->
</div>
```

## 3.4 Использование компонента Bitrix (продвинутая версия)

### Вместо ручного вывода можно использовать компонент:

```php
<?$APPLICATION->IncludeComponent(
    "bitrix:catalog.section",
    ".default",
    array(
        "IBLOCK_ID" => 6,
        "SECTION_ID" => "",
        "ELEMENT_SORT_FIELD" => "sort",
        "ELEMENT_SORT_ORDER" => "asc",
        "FILTER_NAME" => "arrFilter",
        "INCLUDE_SUBSECTIONS" => "Y",
        "PAGE_ELEMENT_COUNT" => 20,
        "LINE_ELEMENT_COUNT" => 3,
        "PROPERTY_CODE" => array("BASE_PRICE_PG", "BASE_PRICE_PO", "BLADE_TYPE"),
        "SHOW_PRICE_COUNT" => 1,
        "CACHE_TYPE" => "A",
        "CACHE_TIME" => "3600",
    )
);?>
```

## 3.5 Проверка

### Откройте страницу: `http://ваш-сайт.ru/catalog/`

**✅ Должно быть:**
- Отображаются карточки дверей из инфоблока
- Изображения загружаются
- Цены отображаются
- Ссылки на детальные страницы работают
- Пагинация работает (если товаров > 20)
